Set analog output scaling (LKIF2_SetAnalogScaling)
RC WINAPI LKIF2_SetAnalogScaling(IN int OutNo,IN int InputValue1,IN int OutputVoltage1,
IN int InputValue2,IN int OutputVoltage2);
Description Set the analog output scaling.
Parameter OutNo  Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
InputValue1  Specify input value 1. The value range is -999999 to 999999.
OutputVoltage1  Specify output voltage 1. The value range is -10500 (mV) to 10500 (mV). (The unit is fixed.)
InputValue2  Specify input value 2. The value range is -999999 to 999999.
OutputVoltage2  Specify output voltage 2. The value range is -10500 (mV) to 10500 (mV). (The unit is fixed.)
Return value RC Refer to "Return code list".
Remark The value set to the LK-G5000 Series depends on the minimum display unit setting of the LK-G5000 Series.
Example: If you set 999999 for measured value 1, the value set to the LK-G5000 Series varies as follows:
Minimum display unit = 0.01 mm : 9999.99 mm
Minimum display unit = 0.001 mm : 999.999 mm
Minimum display unit = 0.0001 mm : 99.9999 mm
Minimum display unit = 0.00001 mm : 9.99999 mm
Minimum display unit = 0.1 um :  99999.9 um
Minimum display unit = 0.01 um : 9999.99 um
Minimum display unit = 0.001 um : 999.999 um
