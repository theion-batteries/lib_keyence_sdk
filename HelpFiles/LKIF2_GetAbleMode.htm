Get ABLE (LKIF2_GetAbleMode)
RC WINAPI LKIF2_GetAbleMode(IN int HeadNo,OUT LKIF_ABLEMODE *AbleMode);
Description Obtain the ABLE mode.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
AbleMode  The ABLE mode is contained.
typedef enum {
LKIF_ABLEMODE_AUTO, // Automatic
LKIF_ABLEMODE_MANUAL, // Manual
} LKIF_ABLEMODE;
Return value RC Refer to "Return code list".
