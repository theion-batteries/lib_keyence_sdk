Set median (LKIF2_SetMedian)
RC WINAPI LKIF2_SetMedian(IN int HeadNo,IN LKIF_MEDIAN Median);
Description Set the median.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
Median Specify the median.
typedef enum {
LKIF_MEDIAN_OFF, // OFF
LKIF_MEDIAN_7, // 7-point
LKIF_MEDIAN_15, // 15-point
LKIF_MEDIAN_31, // 31-point
} LKIF_MEDIAN;
Return value RC Refer to "Return code list".
