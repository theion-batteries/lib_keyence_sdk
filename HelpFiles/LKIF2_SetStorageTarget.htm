Set storage target (LKIF2_SetStorageTarget)
RC WINAPI LKIF2_SetStorageTarget(IN int OutNo,IN BOOL OnOff);
Description Set the OUT No. to store data.
Parameter OutNo Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
OnOff Specify whether to store data of this OUT No.
TRUE: Store data, FALSE: Do not store data.
Return value RC Refer to "Return code list".
