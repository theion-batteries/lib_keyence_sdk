Start ABLE calibration (LKIF2_AbleStart)
RC WINAPI LKIF2_AbleStart(IN int HeadNo);
Description Start the ABLE calibration.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
Return value RC Refer to "Return code list".
Note When this function finishes, the controller is left in the communications mode. After you call this function, be sure to call Finish ABLE calibration API or Cancel ABLE calibration API to reset the controller to the normal mode.
