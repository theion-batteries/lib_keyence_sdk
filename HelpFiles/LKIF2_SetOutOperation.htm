Set OUT to be calculated (AVE/MAX/MIN/P-P) (LKIF2_SetOutOperation)
RC WINAPI LKIF2_SetOutOperation(IN int OutNo,IN int AddSub1,IN LKIF_OUTNO TargetOut);
Description Set the OUT No. to be used for the polynomial calculation (AVE/P-P/MAX/MIN) between OUT.
Parameter OutNo  Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
TargetOut Specify the OUT No. to be calculated.
Specify several Nos. by using OR of LKIF_OUTNO members.
typedef enum {               
 LKIF_OUTNO_01 = 0x0001, //OUT1  
 LKIF_OUTNO_02 = 0x0002, //OUT2  
 LKIF_OUTNO_03 = 0x0004, //OUT3  
 LKIF_OUTNO_04 = 0x0008, //OUT4  
 LKIF_OUTNO_05 = 0x0010, //OUT5  
 LKIF_OUTNO_06 = 0x0020, //OUT6  
 LKIF_OUTNO_07 = 0x0040, //OUT7  
 LKIF_OUTNO_08 = 0x0080, //OUT8  
 LKIF_OUTNO_09 = 0x0100, //OUT9  
 LKIF_OUTNO_10 = 0x0200, //OUT10  
 LKIF_OUTNO_11 = 0x0400, //OUT11  
 LKIF_OUTNO_12 = 0x0800, //OUT12  
 LKIF_OUTNO_ALL = 0x0FFF, //All OUTs  
} LKIF_OUTNO;               
Return value RC Refer to "Return code list".
