Get mask (LKIF2_GetMask)
　 　 　
RC WINAPI LKIF2_GetMask(IN int HeadNo,OUT BOOL *OnOff,OUT int *POS1,OUT int *POS2);
　
Description Obtain the mask information.
　
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
OnOff The status whether the mask is turned on or off is contained. ON: TRUE, OFF: FALSE
Pos1 Set mask boundary 1. A value between -999999 and 999999 is contained.
Pos2 Set mask boundary 2. A value between -999999 and 999999 is contained.
　
Return value RC Refer to "Return code list".
