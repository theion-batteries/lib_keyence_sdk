Measured value output (LKIF2_GetCalcDataALL)
RC WINAPI LKIF2_GetCalcDataALL(OUT int *OutNo,OUT LKIF_FLOATVALUE_OUT *CalcData);
Description Obtain the measured values of all OUTs of the number of used OUT.
Parameter OutNo The number of used OUT is contained.
CalcData Specify the pointer to the measured value structure.
(It is user's responsibility for reserving the area.)
typedef enum {
LKIF_FLOATRESULT_VALID, // Valid data
LKIF_FLOATRESULT_RANGEOVER_P, // Over range at positive (+) side
LKIF_FLOATRESULT_RANGEOVER_N, // Over range at negative (-) side
LKIF_FLOATRESULT_WAITING, // Wait for comparator result
LKIF_FLOATRESULT_ALARM, // Alarm
LKIF_FLOATRESULT_INVALID, // Invalid (error, etc.)
} LKIF_FLOATRESULT;
typedef struct {
int OutNo; // OUT No. (0 to 11)
LKIF_FLOATRESULT FloatResult; // Whether the data is valid or not
float Value; //measurement value during LKIF_FLOATRESULT_VALID.
} LKIF_FLOATVALUE_OUT;
Return value RC Refer to "Return code list".
