Display panel check (LKIF2_GetPanel)
RC WINAPI LKIF2_GetPanel(OUT int *UpperDisp,OUT int *LowerDisp);
Description Obtain the OUT Nos. to be shown in the upper and lower windows of the display panel.
Parameter UpperDisp OUT No. shown in the upper window. The value range is 0 to ((the No. of used OUT)-1), and -1 means no display.
LowerDisp OUT No. shown in the lower window. The value range is 0 to ((the No. of used OUT)-1), and -1 means no display.
Return value RC Refer to "Return code list".
