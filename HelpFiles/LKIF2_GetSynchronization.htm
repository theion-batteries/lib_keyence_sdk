Get synchronization (LKIF2_GetSynchronization)
RC WINAPI LKIF2_GetSynchronization(IN int OutNo,OUT BOOL *OnOff);
Description Obtain the synchronization setting.
Parameter OutNo Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
OnOff Whether data is to be stored or not is contained. TRUE: To be stored, FALSE: Not to be stored
Return value RC Refer to "Return code list".
