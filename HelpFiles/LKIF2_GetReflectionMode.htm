Get mounting mode (LKIF2_GetReflectionMode)
RC WINAPI LKIF2_GetReflectionMode(IN int HeadNo,OUT LKIF_REFLECTIONMODE *ReflectionMode);
Description Obtain the mounting mode.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
ReflectionMode The mounting mode (reflection type) is contained.
typedef enum {
LKIF_REFLECTIONMODE_DIFFUSION, // Diffuse reflection
LKIF_REFLECTIONMODE_MIRROR, // Specular reflection
} LKIF_REFLECTIONMODE;
Return value RC Refer to "Return code list".
