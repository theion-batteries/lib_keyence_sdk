Get measurement type (LKIF2_GetMeasureType)
RC WINAPI LKIF2_GetMeasureType(IN int OutNo,OUT LKIF_MEASURETYPE *MeasureType);
Description Obtain the measurement type.
Parameter OutNo Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
MeasureType The measurement type is contained.
typedef enum {
LKIF_MEASURETYPE_DISPLACEMENT, // Displacement
LKIF_MEASURETYPE_SPEED, // Velocity
LKIF_MEASURETYPE_ACCELERATION ,// Acceleration
} LKIF_MEASURETYPE;
Return value RC Refer to "Return code list".
