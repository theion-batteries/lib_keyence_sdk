Get target surface (LKIF2_GetCalcTarget)
RC WINAPI LKIF2_GetCalcTarget(IN int OutNo,OUT LKIF_CALCTARGET *CalcTarget);
Description Obtain the surface to be measured.
Parameter OutNo  Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
CalcTarget The calculation method is contained.
typedef enum {
LKIF_CALCTARGET_PEAK_1, // Peak 1
LKIF_CALCTARGET_PEAK_2, // Peak 2
LKIF_CALCTARGET_PEAK_3, // Peak 3
LKIF_CALCTARGET_PEAK_4, // Peak 4
LKIF_CALCTARGET_PEAK_1_2, // Peak 1 - Peak 2
LKIF_CALCTARGET_PEAK_1_3, // Peak 1 - Peak 3
LKIF_CALCTARGET_PEAK_1_4, // Peak 1 - Peak 4
LKIF_CALCTARGET_PEAK_2_3, // Peak 2 - Peak 3
LKIF_CALCTARGET_PEAK_2_4, // Peak 2 - Peak 4
LKIF_CALCTARGET_PEAK_3_4, // Peak 3 - Peak 4
} LKIF_CALCTARGET;
Return value RC Refer to "Return code list".
