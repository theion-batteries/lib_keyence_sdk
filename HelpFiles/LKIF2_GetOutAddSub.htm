Get OUT to be calculated (ADD/SUB) (LKIF2_GetOutAddSub)
RC WINAPI LKIF2_GetOutAddSub(IN int OutNo,OUT int *AddSub1,OUT int *AddSub2);
Description Obtain the OUT No. to be used for the addition/subtraction calculation between OUT.
Parameter OutNo  Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
AddSub1 The OUT to which another OUT will be added (from which another OUT will be subtracted) is contained.
AddSub2 The OUT to be added (subtracted) is contained.
Return value RC Refer to "Return code list".
