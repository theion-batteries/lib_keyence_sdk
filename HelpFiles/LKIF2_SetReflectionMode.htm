Set mounting mode (LKIF2_SetReflectionMode)
RC WINAPI LKIF2_SetReflectionMode(IN int HeadNo,IN LKIF_REFLECTIONMODE ReflectionMode);
Description Set the mounting mode.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
ReflectionMode Specify the mounting mode (reflection type).
typedef enum {
LKIF_REFLECTIONMODE_DIFFUSION, // Diffuse reflection
LKIF_REFLECTIONMODE_MIRROR, // Specular reflection
} LKIF_REFLECTIONMODE;
Return value RC Refer to "Return code list".
