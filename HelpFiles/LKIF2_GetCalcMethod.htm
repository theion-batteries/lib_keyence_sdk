Get calculation method (LKIF2_GetCalcMethod)
RC WINAPI LKIF2_GetCalcMethod(IN int OutNo,OUT LKIF_CALCMETHOD *CalcMethod,OUT int *Head_Out_No);
Description Obtain the calculation method.
Parameter OutNo  Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
CalcMethod The calculation method is contained.
typedef enum {
LKIF_CALCMETHOD_HEAD = 0, // Head
LKIF_CALCMETHOD_OUT, // OUT
LKIF_CALCMETHOD_ADD, // ADD
LKIF_CALCMETHOD_SUB, // SUB
LKIF_CALCMETHOD_AVE, // AVE
LKIF_CALCMETHOD_PP, // P-P
LKIF_CALCMETHOD_MAX, // MAX
LKIF_CALCMETHOD_MIN, // MIN
} LKIF_CALCMETHOD;
Head_Out_No When the calculation method is HEAD or OUT, the head No. or OUT No. is contained.
Return value RC Refer to "Return code list".
Remark When CalcMethod is other than LKIF_CALCMETHOD_HEAD or LKIF_CALCMETHOD_OUT, 
the content of Head_Out_No will be ignored.
