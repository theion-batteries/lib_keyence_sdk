Set scaling (LKIF2_SetScaling)
　 　 　
RC WINAPI LKIF2_SetScaling(IN int OutNo,IN int InputValue1,IN int OutputValue1,
IN int InputValue2,IN int OutputValue2);
　
Description Set the scaling.
　
Parameter OutNo  Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
InputValue1 Specify input value 1. The value range is -999999 to 999999.
OutputValue1  Specify display value 1. The value range is -999999 to 999999.
InputValue2 Specify input value 2. The value range is -999999 to 999999.
OutputValue2 Specify display value 2. The value range is -999999 to 999999.
　
Return value RC Refer to "Return code list".
　
Remark Set the input and output value parameters within the following range:
　
Span = (Display value 2 - Display value 1) / (Input value 2 - Input value 1)
-2.0 <= Span <= 2.0
　
The value set to the LK-G5000 Series depends on the minimum display unit setting of the LK-G5000 Series.
Example: If you set 999999 for the scaling, the value set to the LK-G5000 Series varies as follows:
Minimum display unit = 0.01 mm : 9999.99 mm
Minimum display unit = 0.001 mm : 999.999 mm
Minimum display unit = 0.0001 mm : 99.9999 mm
Minimum display unit = 0.00001 mm : 9.99999 mm
Minimum display unit = 0.1 um :  99999.9 um
Minimum display unit = 0.01 um : 9999.99 um
Minimum display unit = 0.001 um : 999.999 um
