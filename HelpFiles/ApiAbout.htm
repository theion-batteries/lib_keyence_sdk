API Configuration
The commands of this DLL are broadly divided into the following four categories:
Measurement control command APIs for controlling measurement. Data can be exchanged when the controller is in "Normal mode".
Setting change command APIs for changing settings. Data can be exchanged when the controller is in "Communications mode".
Setting confirmation command APIs for obtaining the current settings. Data can be exchanged when the controller is in "Communications mode".
Mode change command APIs for changing from "Normal mode" to "Communications mode" and vice versa.
