Get mutual interference prevention (LKIF2_GetMutualInterferencePrevention)
RC WINAPI LKIF2_GetMutualInterferencePrevention(OUT LKIF_MUTUAL_INTERFERENCE_PREVENTION *OnOff);
Description Obtain the mutual interference prevention setting.
Parameter OnOff The mutual interference prevention setting is contained.
typedef enum {
LKIF_MUTUAL_INTERFERENCE_PREVENTION_OFF, // OFF
LKIF_MUTUAL_INTERFERENCE_PREVENTION_AB_ON, // AB-ON
LKIF_MUTUAL_INTERFERENCE_PREVENTION_ABC_ON, // ABC-ON
} LKIF_MUTUAL_INTERFERENCE_PREVENTION;
Return value RC Refer to "Return code list".
