Get received light waveform (LKIF2_GetLight)
RC WINAPI LKIF2_GetLight(IN int HeadNo,IN int PeekNo,OUT int *MeasurePosition,OUT BYTE *WaveData);
Description Obtain the data of the received light waveform of the specified head No.
Parameter HeadNo  Specify the head No. to obtain data.
PeekNo  Specify peak No. when "Transparent object 2" is selected.
MeasurePosition The measurement position for each peak No. is returned.
The area of four data pieces is required.
WaveData A pointer to the area which contains the received light data.
The area of 2048 data pieces is required.
Return value RC Refer to "Return code list".
