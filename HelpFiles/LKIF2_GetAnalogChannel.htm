Get analog output channel (LKIF2_GetAnalogChannel)
RC WINAPI LKIF2_GetAnalogChannel(IN int ChNo,OUT int *OutNo);
Description Obtain the analog output channel.
Parameter ChNo Specify the analog output channel No. The value range is 0 to ((the No. of used analog output channels)-1).
OutNo the OUT No. is contained.
Return value RC Refer to "Return code list".
