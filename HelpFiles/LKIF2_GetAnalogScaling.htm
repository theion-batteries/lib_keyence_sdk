Get analog output scaling (LKIF2_GetAnalogScaling)
RC WINAPI LKIF2_GetAnalogScaling(IN int OutNo,OUT int *InputValue1,OUT int *OutputVoltage1,
OUT int *InputValue2,OUT int *OutputVoltage2);
Description Obtain the analog output scaling setting values.
Parameter OutNo  Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
InputValue1  Input value 1 is contained. The value range is -999999 to 999999.
OutputVoltage1  Output voltage 1 is contained. The value range is -10500 (mV) to 10500 (mV). (The unit is fixed.)
InputValue2  Input value 2 is contained. The value range is -999999 to 999999.
OutputVoltage2  Output voltage 2 is contained. The value range is -10500 (mV) to 10500 (mV). (The unit is fixed.)
Return value RC Refer to "Return code list".
Remark The value contained in the parameter depends on the minimum display unit setting of the LK-G5000 Series.
Example: If the LK-G5000 Series is set as follows, "999999" to be contained represents the following values:
Minimum display unit = 0.01 mm : 9999.99 mm
Minimum display unit = 0.001 mm : 999.999 mm
Minimum display unit = 0.0001 mm : 99.9999 mm
Minimum display unit = 0.00001 mm : 9.99999 mm
Minimum display unit = 0.1 um :  99999.9 um
Minimum display unit = 0.01 um : 9999.99 um
Minimum display unit = 0.001 um : 999.999 um
