Get mutual interference prevention group (LKIF2_GetMutualInterferencePreventionGroup)
RC WINAPI LKIF2_GetMutualInterferencePreventionGroup(IN int HeadNo,
OUT LKIF_MUTUAL_INTERFERENCE_PREVENTION_GROUP *Group);
Description Obtain the mutual interference prevention group setting.
Parameter HeadNo Specify the head No. The value range is 0 to ((the No. of used head)-1).
Group The mutual interference prevention group setting is contained.
typedef enum {
LKIF_MUTUAL_INTERFERENCE_PREVENTION_GROUP_A, // Group A
LKIF_MUTUAL_INTERFERENCE_PREVENTION_GROUP_B, // Group B
LKIF_MUTUAL_INTERFERENCE_PREVENTION_GROUP_C, // Group C
} LKIF_MUTUAL_INTERFERENCE_PREVENTION_GROUP;
Return value RC Refer to "Return code list".
