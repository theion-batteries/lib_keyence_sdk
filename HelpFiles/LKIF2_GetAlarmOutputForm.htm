Get alarm output form (LKIF2_GetAlarmOutputForm)
RC WINAPI LKIF2_GetAlarmOutputForm(OUT LKIF_ALARM_OUTPUT_FORM LKIF_ALARM_OUTPUT_FORM *AlarmOutputForm);
Description Obtain the alarm output form.
Parameter AlarmOutputForm The alarm output form is contained.
typedef enum {
LKIF_ALARM_OUTPUT_FORM_SYSTEM, // System alarm only
LKIF_ALARM_OUTPUT_FORM_MEASURE, // Measured value alarm only
LKIF_ALARM_OUTPUT_FORM_BOTH, // Both system alarm and measured value alarm
} LKIF_ALARM_OUTPUT_FORM;
Return value RC Refer to "Return code list".
