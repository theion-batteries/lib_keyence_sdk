Set measurement mode (LKIF2_SetMeasureMode)
RC WINAPI LKIF2_SetMeasureMode(IN int HeadNo,IN LKIF_MEASUREMODE MeasureMode);
Description Set the measurement mode.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
MeasureMode  Specify the measurement mode.
typedef enum {
LKIF_MEASUREMODE_NORMAL, // Normal
LKIF_MEASUREMODE_HALF_T, // Translucent object
LKIF_MEASUREMDOE_TRAN_1, // Transparent object
LKIF_MEASUREMODE_TRAN_2, // Transparent object 2
LKIF_MEASUREMODE_MRS,
LKIF_MEASUREMODE_OPAQUE
=LKIF_MEASUREMODE_MRS , // Semi opaque object
} LKIF_MEASUREMODE;
Return value RC Refer to "Return code list".
