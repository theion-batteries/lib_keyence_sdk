Set data storage (LKIF2_SetDataStorage)
　 　 　
RC WINAPI LKIF2_SetDataStorage(IN int StorageNum,IN LKIF_STORAGECYCLE StorageCycle);
　
Description Set the data storage (No. of stored data pieces, storage cycle).
　
Parameter StorageNum Specify the No. of data pieces to be stored. The value range is 1 to 1200000.
The maximum number depends on the data to be stored.
StorageCycle  Specify the storage cycle.
　
typedef enum {
LKIF_STORAGECYCLE_1, // Sampling cycle x 1
LKIF_STORAGECYCLE_2, //Sampling cycle x 2
LKIF_STORAGECYCLE_5, // Sampling cycle x 5
LKIF_STORAGECYCLE_10, // Sampling cycle x 10
LKIF_STORAGECYCLE_20, // Sampling cycle x 20
LKIF_STORAGECYCLE_50, // Sampling cycle x 50
LKIF_STORAGECYCLE_100, // Sampling cycle x 100
LKIF_STORAGECYCLE_200, // Sampling cycle x 200
LKIF_STORAGECYCLE_500, // Sampling cycle x 500
LKIF_STORAGECYCLE_1000, // Sampling cycle x 1000
LKIF_STORAGECYCLE_TIMING  // Timing sync
} LKIF_STORAGECYCLE;
　
Return value RC Refer to "Return code list".
