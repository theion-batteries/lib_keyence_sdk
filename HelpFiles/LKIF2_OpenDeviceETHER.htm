Open Ethernet device (LKIF2_OpenDeviceETHER)
RC WINAPI LKIF2_OpenDeviceETHER(IN LKIF_OPENPARAM_ETHERNET *OpenParam);
Description Open the Ethernet device.
Parameter OpenParam Parameter for opening. Specify the value for the structure below.
typedef struct {
 IN_ADDR  IPAddress; //IP address
} LKIF_OPENPARAM_ETHERNET;
* "IN_ADDR" is a structure provided by Microsoft as standard (Refer to "Winsock.h".).
Return value RC Refer to "Return code list".
