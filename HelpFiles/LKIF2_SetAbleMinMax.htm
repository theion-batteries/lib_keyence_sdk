Set ABLE control range (LKIF2_SetAbleMinMax)
RC WINAPI LKIF2_SetAbleMinMax(IN int HeadNo,IN int Min,IN int Max);
Description Set the range of the ABLE control.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
Min  Specify the minimum ABLE value. The value range is 1 to 99.
Max  Specify the maximum ABLE value. The value range is 1 to 99.
Return value RC Refer to "Return code list".
