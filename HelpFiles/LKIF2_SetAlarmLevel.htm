Set alarm level (LKIF2_SetAlarmLevel)
RC WINAPI LKIF2_SetAlarmLevel(IN int HeadNo,IN int AlarmLevel);
Description Set the alarm level.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
AlaramLevel  Specify the alarm level. The value range is 0 to 9.
Return value RC Refer to "Return code list".
