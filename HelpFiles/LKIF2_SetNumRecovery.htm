Set number of times of alarm recovery (LKIF2_SetNumRecovery)
RC WINAPI LKIF2_SetNumRecovery(IN int HeadNo,IN int RecoveryNum);
Description Set the number of times of alarm recovery.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
RecoveryNum Specify the number of times of alarm recovery. The value range is 0 to 9999.
Return value RC Refer to "Return code list".
