For the users of the LK-G3000 DLL
　
  This communication DLL is designed to ensure a certain level of compatibility with the function interfaces of the LK-G3000 DLL so that the users of the LK-G3000 DLL can use the features of the LK-G5000 smoothly.
  Note, however, that some operations may not be exactly the same due to the functional differences between the LK-G3000 and LK-G5000.
The major differences are as follows:
1. The filter type setting and the number of moving average/cutoff frequency setting
  The LK-G3000 DLL can use the filter type setting function "LKIF_SetFilterMode", the number of moving average setting function "LKIF_SetAverage",and the cutoff frequency setting function "LKIF_SetCutOffFrequency" independently.
  With this communication DLL, however, these functions cannot be used independently. You need to specify the filter type first,and then set the number of moving average or cutoff frequency.
2. Functions only available with the LK-G3000
  The statistics functions and analog-through are the functions only available with the LK-G3000 and cannot be used with this communication DLL.
  Calling these functions results in an error.
3. Communication with the LK-G3000
  Although this communication DLL is compatible with the interfaces of the LK-G3000 DLL, it cannot communicate with the LK-G3000.
  It can be used for the communication with the LK-G5000 only.
