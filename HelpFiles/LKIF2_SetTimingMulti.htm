Timing ON/OFF (LKIF2_SetTimingMulti)
RC WINAPI LKIF2_SetTimingMulti(IN LKIF_OUTNO OutNo,IN BOOL OnOff);
Description Set the timing of more than one specified OUT Nos. to on or off.
Parameter OutNo  Specify the OUT Nos. you want to control. 
To specify several Nos., use OR of the LKIF_OUTNO members.
typedef enum {
LKIF_OUTNO_01 = 0x0001, // OUT1
LKIF_OUTNO_02 = 0x0002, // OUT2
LKIF_OUTNO_03 = 0x0004, // OUT3
LKIF_OUTNO_04 = 0x0008, // OUT4
LKIF_OUTNO_05 = 0x0010, // OUT5
LKIF_OUTNO_06 = 0x0020, // OUT6
LKIF_OUTNO_07 = 0x0040, // OUT7
LKIF_OUTNO_08 = 0x0080, // OUT8
LKIF_OUTNO_09 = 0x0100, // OUT9
LKIF_OUTNO_10 = 0x0200, // OUT10
LKIF_OUTNO_11 = 0x0400, // OUT11
LKIF_OUTNO_12 = 0x0800, // OUT12
LKIF_OUTNO_ALL = 0x0FFF, // All OUTs
} LKIF_OUTNO;
OnOff Specify the timing to on (TRUE) or off (FALSE).
Return value RC Refer to "Return code list".
