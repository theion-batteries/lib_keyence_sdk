Set analog output channel (LKIF2_SetAnalogChannel)
RC WINAPI LKIF2_SetAnalogChannel(IN int ChNo,IN int OutNo);
Description Set the analog output channel.
Parameter ChNo Specify the analog output channel No. The value range is 0 to ((the No. of used analog output channels)-1).
OutNo Specify the OUT No. The value range is 0 to ((the No. of used OUT)-1).
Return value RC Refer to "Return code list".
