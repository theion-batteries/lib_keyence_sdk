Set range (LKIF2_SetRange)
RC WINAPI LKIF2_SetRange(IN int HeadNo,IN LKIF_RANGE Range);
Description Set the measurement range.
Parameter HeadNo  Specify the head No. The value range is 0 to ((the No. of used head)-1).
Range Specify the measurement range.
typedef enum {
LKIF_RANGE_CENTER, // CENTER
LKIF_RANGE_FAR, // FAR
} LKIF_RANGE;
Return value RC Refer to "Return code list".
